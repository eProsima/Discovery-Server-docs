Version 2.0.0
#############

This version mainly updates the validation system to adapt it to the new version of the Discovery Server discovery
mechanism released in
`eProsima Fast DDS 2.0.2 <https://fast-dds.docs.eprosima.com/en/latest/notes/notes.html#version-2-0-2>`__.

* Updated the entire test suite.
* Validation system based on expected and fixed test results.
* Removed the snapshots validation mode since the discovery databases of the participants are unique.
